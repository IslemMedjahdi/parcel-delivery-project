# Project Name

## Setup

1. Create an `.env` file in the root directory with the following schema:

   ```plaintext
   CONTRACT_ADDRESS=
   PRIVATE_KEY=
   API_URL=
   ```

2. Run the following commands:

   ```bash
   npm install
   npm run compile
   npm run node
   npm run deploy
   ```

3. Fill in the `.env` file with the required values.

4. Start coding by running:

   ```bash
   npm run dev
   ```
